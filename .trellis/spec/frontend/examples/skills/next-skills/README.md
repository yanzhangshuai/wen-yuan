# Next.js Skills 示例

来源：[vercel-labs/next-skills](https://skills.sh/vercel-labs/next-skills)

本目录包含面向文渊项目的 Next.js 最佳实践代码示例，涵盖：

- `page-with-cache.tsx.template` — 混合 Static + Cached + Dynamic 内容的页面模板
- `server-action.ts.template` — 带缓存失效的 Server Action 模板
- `error-boundary.tsx.template` — 路由级错误边界模板
- `route-handler.ts.template` — API 路由处理器模板

所有 `.template` 文件可直接复制并去掉 `.template` 后缀使用。
