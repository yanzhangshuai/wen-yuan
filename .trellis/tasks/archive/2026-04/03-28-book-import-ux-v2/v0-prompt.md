# v0.app 提示词 — 书籍导入向导页面

> 粘贴以下内容到 [v0.dev](https://v0.dev) 生成 UI 原型。

---

用 Next.js + Tailwind CSS + shadcn/ui 实现一个书籍导入四步向导页面（管理后台）。

## 整体布局

页面最大宽度 `max-w-4xl`，居中展示，底部留白。顶部有标题行：左侧"导入书籍"大标题 + 副标题"按照向导完成书籍导入与解析配置"，右侧"返回书库列表"链接。

标题下方是水平步骤条（Stepper）：

- 共 4 步，均匀分布，步骤圆圈用 `primary` 色填充已完成/当前步骤，未到达步骤用 `border` 色描边
- 已完成步骤显示对勾图标（CheckCircle2），当前及未到达步骤显示数字
- 步骤圆圈之间有连接线，连接线根据当前步骤用动画过渡填充 `primary` 色
- 四步标签：**上传信息 / 确认章节 / 解析配置 / 实时进度**

---

## 第一步：上传信息

卡片（Card）包裹，标题"书籍信息与上传"，副标题"支持 .txt 格式，建议 UTF-8 编码"。

内部是两列表单（`grid grid-cols-2 gap-6`）：

**文件上传区（占满两列）：**
- 大号虚线拖拽区域，内含上传云图标（UploadCloud）+ "点击选择或拖拽文件" + "最大 50MB" 说明
- 选中文件后显示文件名
- 文件不合法时边框变红，下方显示错误文案

**元数据输入（各占一列）：**
- 书名（占位符：自动识别，可修改）
- 作者（占位符：例如：曹雪芹）
- 朝代（占位符：例如：清代）
- 简介（占位符：可选）

**底部按钮（靠右）：**"下一步：生成章节预览"，未选择文件时禁用；提交中显示旋转加载图标。

---

## 第二步：确认章节

卡片标题"章节预览确认"，副标题显示《书名》— 请确认 AI 识别的章节是否正确。

**加载中状态：** 居中旋转圆圈（Loader2）+ "正在识别章节结构..."。

**加载失败状态：** 红色 XCircle 图标 + 错误信息 + "重试"outline 按钮。

**加载成功状态：**

章节列表表格（最大高度 `max-h-100`，超出滚动）：
| 序 | 标题 | 字数 |
|---|---|---|
- 单元格：序号（muted-foreground）、标题（font-medium）、字数（右对齐、半透明）

表格下方一行：左侧"共识别到 N 个章节"，右侧"确认章节，下一步"按钮；确认过程中显示旋转图标。

---

## 第三步：解析配置

**模型选择区：** 标签"选择模型"，下方为响应式卡片网格（移动端 1 列 / PC 端 3 列）。

每个模型卡片：
- 模型名称（粗体）
- 提供商（muted-foreground 小字）
- 底部：modelId 的 outline Badge + 若为默认模型则显示"默认" Badge
- 选中状态：`border-primary` 边框 + `bg-primary/10` 背景；未选中：`border-border`
- hover 时背景轻微变色

**解析范围区（二列布局）：**

左列下拉选框，选项：
- 全书解析
- 多选指定章节
- 指定范围

右列根据所选范围动态渲染：
- **指定范围**：两个数字输入框并排（起始回、结束回）
- **多选指定章节**：在右列显示可滚动的章节复选列表（`max-h-50`），每行 checkbox + 章节标题；选中行有 `bg-primary/5` 背景
- **全书解析**：右列留空

范围为"多选指定章节"时，显示"已选 N / M 个章节"提示文字。

**底部：** 大号"启动解析任务"按钮靠右，模型加载中或未选模型时禁用；启动中显示"启动中..."+ 旋转图标。

---

## 第四步：实时进度

### 总体进度卡片

标题左侧图标动态变化：
- 解析中：蓝色旋转 Loader2
- 完成：绿色 CheckCircle2
- 出错：红色 XCircle

标题文字："解析进行中 / 解析完成 / 解析出错"

副标题：当前阶段文字 + "— XX%"进度百分比

进度条：`w-full bg-muted rounded-full h-2`，内部填充条用 CSS `width` 过渡动画（`transition-all duration-700`）；出错时填充条为红色。

若有错误日志：红色小字显示在进度条下方。

---

### 章节状态表格卡片

标题"章节解析状态"（`text-base`）

表格（可滚动，最大高度 `max-h-100`）：
| 章 | 标题 | 状态 |
|---|---|---|

状态列样式（靠右显示）：

| 状态值 | 显示样式 |
|--------|---------|
| PENDING | 灰色 `○ 等待中`（muted-foreground，小字） |
| PROCESSING | 蓝色旋转 Loader2 + `解析中`（蓝色小字） |
| SUCCEEDED | 绿色 CheckCircle2 + `完成`（绿色小字） |
| FAILED | 红色 XCircle + `失败`（red/destructive 小字） |

---

### 操作按钮区

仅在状态为 COMPLETED 或 ERROR 时出现，靠右排列：
- "返回书库列表" outline 按钮
- "查看书籍详情" primary 按钮

---

## 视觉风格要求

- 整体风格：简洁管理后台，中性色调，primary 为品牌强调色
- 使用 shadcn/ui 组件：Card、Button、Badge、Input
- 图标库：lucide-react（Loader2、CheckCircle2、XCircle、UploadCloud）
- 过渡动画：步骤条进度线、进度条填充均使用 CSS transition
- 响应式：移动端单列，PC 端多列
- 中文界面，所有提示文字、按钮、标签均为中文
