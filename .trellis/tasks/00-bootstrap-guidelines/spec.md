# Feature Spec - Spec-Kit + Trellis 中文流程体系完善

## 背景
当前团队已经有 Trellis 与 Spec-Kit 的基础规范，但需要中文化、流程化、案例化，并提供可快速执行的记忆入口。

## 目标
为团队提供一套可直接使用的中文文档体系，覆盖：
1. 规范总览
2. 详细使用流程
3. 一页决策树
4. 泳道图可视化
5. 三个预设触发口令（flow-lite/flow-feature/flow-bug）

## 需求范围
- 在 `.trellis/spec/guides/` 下新增或补充中文文档。
- 将新增文档加入 `guides/index.md` 导航。
- 文档中补充“命令说明（完整）”与典型使用场景。

## 验收标准
- [ ] 中文流程文档可独立阅读并执行。
- [ ] 至少包含 3 个任务案例（小改动/功能开发/问题修复）。
- [ ] 包含 Mermaid 版泳道图。
- [ ] 包含三个触发口令及映射流程。
- [ ] `guides/index.md` 可发现所有新增文档。
