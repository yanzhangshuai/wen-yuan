# OpenSpec Constitution

## Non-Negotiable Principles
1. Evidence First: relation/event conclusions must include source evidence ids.
2. Human in the Loop: reviewed data overrides auto-generated outputs.
3. Contract Stability: API error code and payload shape are machine-stable.
4. Reproducibility: prompts/model versions/task traces are auditable.
5. Safety over coverage: low-confidence outputs are draft/review, not facts.

## Delivery Gates
- Success/failure/boundary verification must be present in check evidence.
- Multi-entity writes use transactions.
- Cross-layer contract changes require strict strategy.
