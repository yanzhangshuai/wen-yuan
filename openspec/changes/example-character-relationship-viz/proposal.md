# 功能提案：角色关系可视化

## 背景

用户需要直观地理解小说中角色之间的关系网络，当前只有文本列表形式，缺乏可视化展示。

## 业务价值

1. 提升用户体验：图形化展示比文本列表更直观
2. 发现隐藏关系：通过网络图可以发现间接关系
3. 辅助分析决策：帮助研究者快速把握角色关系全局

## 目标用户

- 文学研究者：需要分析角色关系网络
- 普通读者：想要快速了解角色关系

## 功能范围

### 包含
- 角色关系网络图展示
- 关系类型筛选（亲属、朋友、敌对等）
- 点击节点查看角色详情
- 点击边查看关系证据

### 不包含
- 关系强度动态计算（留待 M2）
- 时间轴演进动画（留待 M2）
- 多小说关系对比（留待 M3）

## 验收标准

1. 能够展示至少 20 个角色的关系网络
2. 支持按关系类型筛选
3. 点击节点/边能正确跳转到详情
4. 页面加载时间 < 3s

## 风险与依赖

- 依赖：角色关系数据已存在于数据库
- 风险：大量节点时性能可能下降（需要分页或聚合）
