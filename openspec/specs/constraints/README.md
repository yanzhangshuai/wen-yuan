# 产品约束

本目录定义 Wen-Yuan 项目的产品约束和边界。

## 核心约束

### 1. MVP 优先
- 聚焦《儒林外史》单一小说
- 先验证核心价值，再扩展功能
- 避免过度设计

### 2. 证据可追溯
- 所有 AI 结论必须关联原文证据
- 证据 ID 可定位到具体文本片段
- 支持证据链的完整展示

### 3. 人工审核为最终权威
- AI 分析结果为建议，非最终结论
- 人工审核后的数据覆盖 AI 输出
- 审核历史可追溯

### 4. 低成本交付
- 优先使用开源技术栈
- 控制 AI API 调用成本
- 避免复杂的基础设施依赖

### 5. 安全与隐私
- 不存储敏感个人信息
- API 访问需要认证
- 数据访问遵循最小权限原则

## 技术约束

- 前端：Next.js App Router + React
- 后端：Next.js Server Actions + API Routes
- 数据库：PostgreSQL + Prisma
- AI：OpenAI API（可配置其他提供商）

## 性能约束

- 页面首次加载 < 3s
- API 响应时间 < 2s（不含 AI 调用）
- 支持 100+ 章节的小说分析
